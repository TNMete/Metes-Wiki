# Gesetze der Boolschen Algebra

### 1. Kommutativgesetz:

``A ∨ B = B ∨ A``

``A ∧ B = B ∧ A``

### 2. Assoziativgesetz:

``(A ∨ B) ∨ C = A ∨ (B ∨ C)``

``(A ∧ B) ∧ C = A ∧ (B ∧ C)``

### 3. Distributivgesetz:

``A ∨ (B ∧ C) = (A ∨ B) ∧ (A ∨ C)``

``A ∧ (B ∨ C) = (A ∧ B) ∨ (A ∧ C)``

### 4. Identitätsgesetz:

``A ∨ 0 = A``

``A ∧ 1 = A``

### 5. Null- und Einsgesetz:

``A ∨ 1 = 1``

``A ∧ 0 = 0``

### 6. Negationsgesetz:

``A ∨ ¬A = 1``

``A ∧ ¬A = 0``

### 7. Idempotenzgesetz:

``A ∨ A = A``

``A ∧ A = A``

### 8. Absorptionsgesetz:

``A ∨ (A ∧ B) = A``

``A ∧ (A ∨ B) = A``

### 9. De-Morgansches Gesetz:

``¬(A ∧ B) = ¬A ∨ ¬B``

``¬(A ∨ B) = ¬A ∧ ¬B``
